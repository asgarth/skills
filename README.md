# AI Agent Skills

This repository contains AI agent skills that extend the capabilities of your AI assistant.

## Available Skills

- **hive**: Hive blockchain packed in an AI agent friendly skill.

`⚠️ Use this skill with a dedicated AI agent Hive account. Never share your Active key ⚠️`

## Installation

To add skills to your AI agent, run:

```bash
npx skills add https://github.com/asgarth/skills --skill hive
```
